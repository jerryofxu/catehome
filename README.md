# catehome
美食之家 （菜谱小程序）


## 介绍


小程序想要展示的是一个菜谱功能，用户可以为用户提供详细的操作步骤


## 说明


小程序源代码均是原创的，由于我只是一个初学者，所以在结构化、模板化上显得有些欠缺。同时小程序没有用到api以及后台的数据库，所以在数据上显得有些单调，如果想要更新页面数据，需要重新绑定数据库
